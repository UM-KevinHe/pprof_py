pprof_py.utils.setup_logger

pprof_py.utils.setup_logger(name: str, level=20, log_to_file: bool = False, log_dir: str | None = None, time_zone: str = 'US/Eastern') → Logger[source]

Setup a logger with the given name and level.

Parameters: - name (str): The name of the logger. - level (int): The logging level. Default is logging.INFO. - log_to_file (bool): If True, logs will also be written to a file. Default is False. - log_dir (str): Directory where the log file will be saved. Default is None (current working directory). - time_zone (str): The time zone for the timestamps. Default is “US/Eastern”.

Returns: - logging.Logger: Configured logger.
