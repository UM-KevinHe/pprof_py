pprof_py.plotting.plot_caterpillar(df: DataFrame, estimate_col: str = 'estimate', ci_lower_col: str | None = 'lower', ci_upper_col: str | None = 'upper', group_col: str | None = None, flag_col: str | None = None, labels: List[str] = ['Lower', 'Expected', 'Higher'], sort_by_estimate: bool = True, point_size: float = 2.0, point_color_default: str = '#56B4E9', flag_colors: Dict[int, str] = {-1: '#E69F00', 0: '#56B4E9', 1: '#009E73'}, point_alpha: float = 0.8, line_size: float = 0.8, refline_value: float | None = 0.0, refline_linestyle: str = '--', refline_color: str = 'black', font_size: float = 12, tick_label_size: float = 10, errorbar_size: float = 0.5, errorbar_alpha: float = 0.5, errorbar_color_default: str = 'grey', figure_size: Tuple[float, float] = (8, 6), plot_title: str = 'Caterpillar Plot', xlab: str = 'Estimate', ylab: str = 'Group', legend_location: str = 'best', orientation: Literal['vertical', 'horizontal'] = 'vertical', add_grid: bool = True, grid_style: str = ':', grid_alpha: float = 0.6, remove_top_right_spines: bool = True, save_path: str | None = None, dpi: int = 300) → None[source]ÔÉÅ

Create a highly customizable caterpillar plot from a DataFrame.

Plots estimates on the x-axis against sorted groups on the y-axis, with optional confidence intervals and color-coding based on flags.

Parameters:

  • df (pd.DataFrame) ‚Äì DataFrame containing the data to plot. Index should represent groups if group_col is None.

  • estimate_col (str, default='estimate') ‚Äì Column name for point estimates (plotted on x-axis).

  • ci_lower_col (str or None, default='lower') ‚Äì Column name for lower confidence interval bounds. If None, no error bars.

  • ci_upper_col (str or None, default='upper') ‚Äì Column name for upper confidence interval bounds. If None, no error bars.

  • group_col (str or None, default=None) ‚Äì Column name for group identifiers. If None, uses the DataFrame index.

  • flag_col (str or None, default=None) ‚Äì Column name for flags (e.g., -1, 0, 1). If provided, points and error bars are colored based on flag values using flag_colors.

  • labels (List[str], default=["Lower", "Expected", "Higher"]) ‚Äì Labels for flag categories (-1, 0, 1) used in the legend.

  • sort_by_estimate (bool, default=True) ‚Äì Whether to sort the groups on the y-axis by estimate values.

  • point_size (float, default=2.0) ‚Äì Scaling factor for marker size (base size is 30).

  • point_color_default (str, default="#56B4E9") ‚Äì Default color for points if flag_col is None or a flag value is missing from flag_colors.

  • flag_colors (Dict[int, str], default={-1: "#E69F00", 0: "#56B4E9", 1: "#009E73"}) ‚Äì Dictionary mapping flag values to colors.

  • point_alpha (float, default=0.8) ‚Äì Transparency of points.

  • line_size (float, default=0.8) ‚Äì Thickness for the reference line.

  • refline_value (float or None, default=0.0) ‚Äì Value for the vertical reference line (on the estimate axis). If None, no line.

  • refline_linestyle (str, default='--') ‚Äì Line style for the reference line.

  • refline_color (str, default="black") ‚Äì Color for the reference line.

  • font_size (float, default=12) ‚Äì Base font size for labels and title.

  • tick_label_size (float, default=10) ‚Äì Font size for axis tick labels.

  • errorbar_size (float, default=0.5) ‚Äì Thickness (linewidth) of the error bars.

  • errorbar_alpha (float, default=0.5) ‚Äì Transparency of the error bars.

  • errorbar_color_default (str, default="grey") ‚Äì Default color for error bars if flag_col is None or a flag value is missing from flag_colors. If flags are used, error bars match point colors.

  • figure_size (Tuple[float, float], default=(8, 6)) ‚Äì Figure size for the plot.

  • plot_title (str, default="Caterpillar Plot") ‚Äì Plot title.

  • xlab (str, default="Estimate") ‚Äì X-axis label.

  • ylab (str, default="Group") ‚Äì Y-axis label.

  • legend_location (str, default='best') ‚Äì Location of the legend if flag_col is used.

  • orientation ("vertical" (groups on y-axis, estimates on x-axis)) ‚Äì or ‚Äúhorizontal‚Äù (groups on x-axis, estimates on y-axis).

  • add_grid (bool, default=True) ‚Äì Whether to display a grid.

  • grid_style (str, default=':') ‚Äì Style of the grid lines.

  • grid_alpha (float, default=0.6) ‚Äì Transparency of the grid lines.

  • remove_top_right_spines (bool, default=True) ‚Äì Whether to remove the top and right spines.

  • save_path (str or None, default=None) ‚Äì Path to save the plot. If None, plot is displayed.

  • dpi (int, default=300) ‚Äì Resolution for saving the plot.
